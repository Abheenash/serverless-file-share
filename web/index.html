Create a share link

Upload a file and get a link that self-destructs on a timer. Encrypted at rest with a customer-managed KMS key; the object is deleted when the link expires.

How it works

  1. The browser requests a presigned upload URL from API Gateway → Lambda and uploads straight to S3.
  2. Metadata is written to DynamoDB with a TTL set to your chosen lifetime.
  3. When the TTL fires, a DynamoDB Stream triggers a reaper Lambda that deletes the S3 object.
  4. An S3 lifecycle rule is the backstop — nothing survives past 8 days.
